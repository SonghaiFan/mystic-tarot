# 塔罗牌图片目录

此目录包含所有 78 张塔罗牌的图片（22 张大阿尔卡那 + 56 张小阿尔卡那）。

## 生成方法

运行以下命令从 Sacred Texts Archive 下载所有图片：

```bash
npm run download-cards
```

## 图片命名规则

- **大阿尔卡那**: `ar00.jpg` - `ar21.jpg`
- **权杖牌组**: `wa01.jpg` - `wa14.jpg`
- **圣杯牌组**: `cu01.jpg` - `cu14.jpg`
- **宝剑牌组**: `sw01.jpg` - `sw14.jpg`
- **星币牌组**: `pe01.jpg` - `pe14.jpg`
